Proyectos

Estos son algunos de los proyectos que realicé:

Henry Videogames

Proyecto individual hecho durante el bootcamp de Henry. Sitio web que permite al usuario visualizar, ordenar, filtrar y observar los detalles de diversos videojuegos. Además, presenta la opción de crear un videojuego propio. Fue desarrollada utilizando HTML, CSS, JavaScript, React.js y Redux.

Royal Makeup

Proyecto grupal hecho durante el bootcamp de Henry. E-Commerce que contiene un home, catálogo, perfiles de usuario, carrito de compras, compras con PayPal, servicio de mail y página de administrador. Las tecnologías utilizadas fueron: HTML, CSS, JavaScript, Tailwind CSS, React.js, Material UI, Redux, Sequelize, Express.js, Node.js, PostgreSQL, Nodemailer y Firebase Authentication.

BuscadogQR

Desarrollo de una página web con el objetivo de solucionar la búsqueda de mascotas perdidas. Fue desarrollada usando JavaScript, HTML, CSS, Tailwind CSS, React.js y Firebase.
